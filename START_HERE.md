## ğŸ› Having Issues?

### Common Problems

**"Extension won't load"**
- Make sure you're using Chrome Canary (not regular Chrome)
- Check Developer mode is ON
- Look for errors at chrome://extensions/

**"No transcription"**
- Verify AI flags are enabled
- Check model downloaded at chrome://components/
- Restart Chrome Canary completely
- Grant microphone permissions

**"Audio not captured"**
- Click "Share" when Chrome asks for permissions
- Make sure you're in a Google Meet/Zoom meeting
- Check system audio isn't muted
